## Shared Dependencies

All deployment models require shared access to:

- RabbitMQ
- Redis
- consistent queue configuration
- consistent result TTL and job prefix configuration

## Operational Recommendation

Use the API readiness endpoint for traffic routing, and use worker readiness for rollout validation and worker fleet health.
