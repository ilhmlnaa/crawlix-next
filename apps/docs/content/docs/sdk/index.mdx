---
title: SDK Overview
description: Build Node.js integrations against Crawlix Next with the official SDK package.
---

The Crawlix Next SDK is a Node.js client for the public async jobs API. It wraps the API key flow, job creation, polling, targeted worker dispatch, and webhook signature verification into one package.

## What it covers

- API key authenticated requests
- `POST /api/jobs`
- `GET /api/jobs/:jobId`
- `GET /api/jobs/:jobId/result`
- safe polling helpers
- webhook signature verification

## What it does not cover

- admin session login
- dashboard-only endpoints
- API key management
- retry and cancel admin actions
- worker fleet management

## Package Name

```bash
@crawlix/sdk-node
```
